# Converter 
Эта программа умеет переводить файлы из одного формата в другой, а именно jsonTOcsv и csvTOjson.
# Usage
Проверить работоспособность данного проекта можно 2 способами:
1) с помощью команды:
```python3 setup.py```
2) с помощью команды:\
```python3 setup.py -in <name_file> -out <name_file>```
